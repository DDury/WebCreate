HTML & CSS Basics Summary

HTML Summary

HTML (HyperTextMarkupLanguage) is used to define our page content, structure and meaning.You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content and then a closing tag

  • There's a long list of available elements but you'll gain experience over time, no worries

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page contents

  • Style are assigned via property-value pairs
  • You can assign style via the "style" attribute
  • To avoid code duplication, you typically use global style (e.g. via the "style" element) instead
  • Alternatively, you can also work with external stylesheet files which you "link" to
  • When working with CSS, concepts like "inheritance", "speciality" and the "Box model" should be understood

Learn more about all available CSS property and value on the MDN CSS property reference
